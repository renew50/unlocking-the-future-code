# EP 18-02 Test cases

Test cases
วิธีการคิด test cases แบ่งออกเป็น 3 กลุ่มตามนี้  

1) normal cases คือ cases ปกติที่เกิดขึ้นบ่อยๆ  
2) alternative cases คือ cases ที่เกิดขึ้นนานๆที
3) exception cases คือ กรณีที่มีข้อผิดพลาดเกิดขึ้นในโปรแกรม
